## Settings

Create a `.env` file with the following vars:

```yaml
DOMAIN=
ADMIN_PASSWORD=
CONFIG_PASSWORD=
READONLY_USER_USERNAME=
READONLY_USER_PASSWORD=
```

An example could be:


```yaml
DOMAIN=example.org
ADMIN_PASSWORD=admin
CONFIG_PASSWORD=config
READONLY_USER_USERNAME=readonly
READONLY_USER_PASSWORD=readonly
```
